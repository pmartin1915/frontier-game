#### MED-03: Audio System Memory Leak
**Location:** `src/audio/ambiance.ts`
**Risk:** High - Performance degradation, potential application crashes, and poor user experience over extended play sessions.
**Status:** No progress detected.
**Proposed Action Plan:**
1.  Implement track cleanup with a 5-minute Time-To-Live (TTL).
2.  Add memory monitoring and leak detection tests.
3.  Ensure proper cleanup on system destruction.

```typescript
class AmbianceSystem {
  private static readonly TRACK_TTL_MS = 300000; // 5 minutes
  private _tracks: Map<string, {track: Howl, lastUsed: number}> = new Map();
  private _cleanupInterval: NodeJS.Timeout;

  constructor() {
    this._cleanupInterval = setInterval(() => this.cleanupTracks(), 60000); // Check every minute
  }

  private cleanupTracks(): void {
    const now = Date.now();
    this._tracks.forEach((value, key) => {
      if (now - value.lastUsed > AmbianceSystem.TRACK_TTL_MS) {
        value.track.unload();
        this._tracks.delete(key);
      }
    });
  }

  public destroy(): void {
    clearInterval(this._cleanupInterval); // Stop cleanup interval
    this._tracks.forEach(value => value.track.unload()); // Unload all remaining tracks
    this._tracks.clear();
  }
}
```
