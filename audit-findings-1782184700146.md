#### HIGH-02: Biome/Season Hunting Yield Variation Missing
*Status:* Open
*Location:* `src/systems/supplies.ts`
*Description:* Hunt yields use flat values without biome/season variation, confirmed unimplemented in 2026-06-18 QA report.
*Impact:* Reduced game realism and strategic depth.
*Recommendations:*
```typescript
// Recommended implementation pattern
const HUNTING_YIELD_BY_BIOME = {
  forest: { base: 3, seasonFactor: 1.2 },
  desert: { base: 1, seasonFactor: 0.8 },
  // ...
};

function calculateHuntYield(biome: Biome, dayOfYear: number): number {
  const { base, seasonFactor } = HUNTING_YIELD_BY_BIOME[biome];
  const seasonVariation = 1 + 0.2 * Math.sin(2 * Math.PI * dayOfYear / 365);
  return Math.round(base * seasonFactor * seasonVariation * (0.8 + 0.4 * Math.random()));
}
```
