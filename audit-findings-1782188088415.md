### High Severity (4 Open)

#### HIGH-01: Missing Narrative Thread Escalation
- **Location:** `src/engine/director.ts`
- **Status:** Open
- **Description:** Director system does not advance narrative thread statuses based on game events, limiting dynamic storytelling.
- **Impact:** Reduced player engagement and narrative depth.
- **Recommendation:**
  - Implement thread status advancement logic with clear progression triggers
  - Add event-based triggers for thread progression (e.g., "reach waypoint X", "survive Y days")
  - Create comprehensive test coverage for narrative escalation paths

#### HIGH-02: Biome/Season Hunting Yield Variation Missing
- **Location:** `src/systems/supplies.ts`
- **Status:** Open
- **Description:** Hunt yields use flat values without biome/season variation, reducing strategic depth.
- **Impact:** Compromised game realism and replayability.
- **Recommendation:**
  ```typescript
  // Proposed implementation structure
  const HUNTING_YIELD_BY_BIOME = {
    forest: { base: 3, seasonFactor: 0.3 },
    desert: { base: 1, seasonFactor: 0.1 },
    // ...
  };

  function calculateHuntYield(biome: Biome, dayOfYear: number): number {
    const biomeData = HUNTING_YIELD_BY_BIOME[biome];
    const seasonFactor = 1 + biomeData.seasonFactor * Math.sin(2 * Math.PI * dayOfYear / 365);
    return Math.round(biomeData.base * seasonFactor * (0.8 + Math.random() * 0.4));
  }
  ```

#### HIGH-03: Companion Loyalty Momentum System Incomplete
- **Location:** `src/systems/companions.ts`
- **Status:** Open
- **Description:** Desertion probability calculation lacks loyalty+momentum combination logic.
- **Impact:** Companion interactions lack depth and realism.
- **Recommendation:**
  - Implement momentum tracking for loyalty changes
  - Add desertion probability calculation that considers both current loyalty and recent trends
  - Document as optional feature for post-launch implementation

#### HIGH-04: Type Safety Issues Persist
- **Location:** Agent bridge, audio system
- **Status:** Open
- **Description:** Type assertions could fail at runtime despite previous remediation attempts.
- **Impact:** Potential runtime errors and unpredictable behavior.
- **Recommendation:**
  - Replace all type assertions with proper type guards
  - Implement runtime validation for critical paths
  - Add static analysis checks to prevent regression

### Medium Severity (3 Open)

#### MED-01: Encounter Outcome Randomization Missing
- **Location:** `src/systems/encounters.ts`
- **Status:** Open
- **Description:** Numeric outcomes use fixed values instead of ±10% variation.
- **Impact:** Reduced replayability and encounter unpredictability.
- **Recommendation:**
  ```typescript
  // Proposed implementation
  function applyOutcomeVariation(baseValue: number): number {
    const variation = 0.1 * (Math.random() * 2 - 1); // ±10%
    return Math.round(baseValue * (1 + variation));
  }
  ```

#### MED-02: Companion-Specific Encounter Choices Missing
- **Location:** `src/types/encounters.ts`, `src/systems/encounters.ts`
- **Status:** Open
- **Description:** No `companion: CompanionId` requirement type for encounter choices.
- **Impact:** Limits companion-specific interactions and role-playing depth.
- **Recommendation:**
  - Add companion requirement type to encounter definitions
  - Implement choice availability filtering based on current companions
  - Add test case: "choice requiring Coe is unavailable if Coe deserted"

#### MED-03: Audio System Memory Leak
- **Location:** `src/audio/ambiance.ts`
- **Status:** Open
- **Description:** Uncleaned `_tracks` Map causes memory growth over time.
- **Impact:** Potential performance degradation during long play sessions.
- **Recommendation:**
  ```typescript
  // Proposed solution
  class AmbianceSystem {
    private _tracks: Map<string, Howl> = new Map();
    private _trackTTL: Map<string, number> = new Map();

    public unloadUnusedTracks(): void {
      const now = Date.now();
      this._tracks.forEach((track, key) => {
        if (now - (this._trackTTL.get(key) || 0) > 300000) { // 5 minute TTL
          track.unload();
          this._tracks.delete(key);
          this._trackTTL.delete(key);
        }
      });
    }
  }
  ```
