### 1. Bugs

*   **[HIGH] `src/systems/health.ts` - `daysUntilCritical` not decremented.**
    *   **Description:** The `calculateHealth` function correctly adds new `ActiveCondition` entries with an initial `daysUntilCritical` value. However, it does not contain logic to decrement this `daysUntilCritical` counter for *existing* active conditions each day, nor does it trigger any critical effects when the counter reaches zero. This is explicitly identified as the "Next" task in `ai/STATE.md` and a core mechanic for condition escalation.
    *   **Severity:** High
    *   **Recommendation:** Implement the logic within `calculateHealth` (or a new dedicated function) to iterate through existing `player.conditions` and `horse.conditions`, decrement `daysUntilCritical`, and apply permanent debuffs or emit "worsened" `HealthEvent`s when it reaches 0.
