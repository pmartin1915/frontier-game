### High Severity (4)

#### HIGH-01: Biome/Season Hunting Yield Variation Missing
*Status:* Open
*Location:* `src/systems/supplies.ts`
*Description:* Hunt yields use flat values without biome/season variation, confirmed unimplemented in 2026-06-18 QA report.
*Impact:* Reduces game realism and strategic depth in resource management.
*Recommendations:*
```typescript
// Implementation example:
const HUNTING_YIELD_BY_BIOME = {
  forest: { base: 3, min: 2, max: 4 },
  desert: { base: 1, min: 0, max: 2 },
  // ... other biomes
};

function calculateHuntYield(biome: Biome, dayOfYear: number): number {
  const biomeData = HUNTING_YIELD_BY_BIOME[biome];
  const seasonFactor = 1 + 0.2 * Math.sin(2 * Math.PI * dayOfYear / 365);
  const variation = 0.2 * (Math.random() * 2 - 1); // ±20%
  return Math.round(biomeData.base * seasonFactor * (1 + variation));
}
```
- Add biome-specific yield tables with min/max ranges
- Implement seasonal variation using sinusoidal function
- Add ±20% randomization to yields
- Update all hunting-related tests

#### HIGH-04: Incomplete Companion Loyalty Momentum
*Status:* Open
*Location:* `src/systems/companions.ts`
*Description:* Desertion probability doesn't account for loyalty+momentum combination, confirmed in 2026-06-17 design review.
*Impact:* Companion desertion mechanics lack depth and realism.
*Recommendations:*
```typescript
// Implementation example:
function calculateDesertionProbability(loyalty: number, momentum: number): number {
  const baseProbability = 0.1 * (10 - loyalty) / 10;
  const momentumFactor = 1 + (momentum < 0 ? 0.5 * Math.abs(momentum) : 0);
  return Math.min(0.8, baseProbability * momentumFactor);
}
```
- Implement combined loyalty+momentum calculation
- Add test cases for various loyalty/momentum combinations
- Document the desertion probability formula
- Consider adding visual indicators for high desertion risk
