#### HIGH-02: Biome/Season Hunting Yield Variation Missing
**Location:** `src/systems/supplies.ts`
**Risk:** Reduces game realism and strategic depth
**Recommendation:**
- Implement biome-specific yield tables (Day 1)
- Add seasonal variation using sinusoidal function (Day 2)
- Implement ±20% randomization (Day 3)
- Create property-based tests for yield calculations

#### HIGH-03: River Crossing Terrain Lock Incomplete
**Location:** `src/systems/movement.ts`, `src/types/game-state.ts`
**Risk:** Breaks core traversal mechanics
**Recommendation:**
- Add `riverCrossingInProgress: boolean` to JourneyState interface
- Modify movement system to block movement when flag is set
- Implement flag clearing on encounter resolution
- Add comprehensive test coverage

#### HIGH-04: Type Safety Issues Persist
**Location:** Agent bridge, audio system
**Risk:** Potential runtime errors and data corruption
**Recommendation:**
- Replace all type assertions with proper type guards
- Implement Zod schema validation for agent bridge
- Add static analysis checks to CI pipeline
- Add runtime validation for critical paths

### Medium Severity Issues (All Overdue)

**Location:** `src/systems/encounters.ts`
**Risk:** Reduces replayability

#### MED-02: Companion-Specific Encounter Choices Missing

**Location:** `src/audio/ambiance.ts`
**Risk:** Potential performance degradation
