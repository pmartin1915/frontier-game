## Systemic Issues

### 1. False-Positive Epidemic
**Root Cause**: Over-reliance on pattern matching without semantic analysis
**Impact**:
- Erodes trust in audit system
- Wastes developer time on triage
- Masks real issues in noise

**Mitigation Strategy**:
1. **Short-term**: Implement a "confidence score" threshold (e.g., discard findings with <70% confidence)
2. **Medium-term**: Add a feedback loop where human triage results train the classification model
3. **Long-term**: Replace keyword-based detection with semantic code analysis

### 2. State Synchronization Failure
**Root Cause**: Audit system operates on stale repository snapshots
**Impact**:
- Findings reference deleted/renamed files
- Misses new files added after snapshot creation

**Mitigation Strategy**:
1. **Immediate**: Add pre-audit `git pull` and file existence check
2. **Process**: Implement incremental audits for changed files only
3. **Monitoring**: Track "stale finding" rate as a KPI
